# NextJS Project Starter

## Getting Started

1. clone this repo
1. run `yarn install`
1. Create `.env.local` (see .env.local.example)
1. run `yarn start` to start the dev server
1. run `yarn test` to run Cypress integration tests
1. visit http://localhost:3000
1. Work!
